# keepclean
Site KeepClean
O que sua página deve conter (mínimos obrigatórios): 
1. Estrutura HTML bem organizada, com cabeçalho, corpo e rodapé(imagens,
navegação entre páginas, etc.). 
2. Uso de CSS para personalizar o visual (cores, fontes, espaçamentos, etc.). 
3. Interatividade com JavaScript, como:
o Clique em botões para mostrar/esconder algo.
o Validação simples de formulário. 
4. Layout visual agradável e de boa legibilidade 
5. Identificação do aluno no rodapé (nome e turma). 
6. Organização de pastas e arquivos.  

Entrega: 
• Envie a pasta completa do projeto (HTML, CSS, JS e possíveis imagens)
compactada em .zip.
